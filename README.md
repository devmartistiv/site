# Vitamove Landing Page

Лендинг для фитнес-приложения Vitamove, созданный на React + Vite.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте http://localhost:5173 в браузере.

## Сборка

```bash
npm run build
```

## Структура

- **Header** - шапка с навигацией
- **Hero** - главный экран с призывом к действию
- **Features** - возможности приложения (6 карточек)
- **Screenshots** - скриншоты интерфейса с мокапами телефонов
- **Download** - секция скачивания с кнопками App Store и Google Play
- **Footer** - подвал с ссылками

## Цветовая схема

- Дополнительный: #4a9fe5 (синий)
